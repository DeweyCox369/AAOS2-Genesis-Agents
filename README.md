# AAOS 2.0 Genesis Agents - MCP Registry

This repository contains the three Genesis Agents for the Autonomous Agent Operating System (AAOS 2.0), configured for publication to the official Model Context Protocol (MCP) Registry.

## Agents

### 1. CFO Agent (aaos2/cfo-agent)
Autonomous Chief Financial Officer for portfolio management, asset allocation, and yield optimization with x402-PRIVATE privacy support.

### 2. Trader Agent (aaos2/trader-agent)
Autonomous Trader for spot trading, arbitrage, and market making with MEV protection via x402.

### 3. Risk Manager Agent (aaos2/risk-agent)
Autonomous Chief Risk Officer for portfolio risk assessment, hedging, and compliance with x402-PRIVATE privacy support.
